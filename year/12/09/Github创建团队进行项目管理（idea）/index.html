    Github创建团队进行项目管理（idea）

    注：在这之前需要在自己电脑下载Git，然后在idea中配置Git

    一，注册github账号

    1.进入github官网

    github网址：https://github.com/

    2.填写信息，创建用户

    3.密码框下面需要验证，下一步

    4.选择需要的计划

    5.验证邮箱地址，在邮箱上完成验证

    6.为自己创建一个仓库

    二，创建团队

    1.右上角加号->New organization

    2.根据自己的需求选择合适的计划

    3.建立团队

    4.创建成功，可以根据用户名和电子邮箱邀请成员

    5.为自己的团队创建仓库管理项目

    6.创建成功，把仓库克隆到自己电脑（这里是在idea上克隆的）

    三，在idea上克隆项目

    1.在idea上克隆项目，根据github仓库url，选择存放路径

    2.克隆成功，就可以在里面写自己的代码了

    注：也可以把自己的源码复制到仓库中，直接在文件夹中进行复制,除了克隆到本地文件(.git)不要修改，把其他项目文件复制到仓库中

    3.将代码更新与下载最新代码

    ​ 通过 git—>add->commit directory->Repository->push 将代码提交到github

    ​ 通过 git->Repository->pull 下载最新代码（其他成员修改之后的代码）

    4.需要登录自己的github进行绑定

    注：把其他团队成员邀请到你的团队之后，就可以把项目克隆到自己的电脑上，进行代码的修改，上传，下载最新代码了。

    注：在这之前需要在自己电脑下载Git，然后在idea中配置Git

    四，在idea中配置git，github
